Code used in windows powershell (admin) to set up the background service:
    sc.exe create arbot Displayname= "arbot" binpath= "srvstart.exe arbot -c C:\arbot.ini" start= auto

<servicename> = arbot # the generic services name
<path to srvstart config file> = C:config_file.ini
<starttype> = auto # automatically starts with windows

srvstart was downloaded from a git repository and place into my C:\
